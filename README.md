# Fracture

Reality-breaking survival. Chunks randomly swap with their parallel dimension counterparts. Your base might suddenly contain hostile architecture. Creatures phase between realities. Trust nothing.

## Why This Exists?

Procedural generation is old news. Fracture makes the world itself unstable. You're not exploring a random map — you're surviving a map that changes while you stand on it. Your safe zone can become lethal without warning. That's not a bug, it's the entire point.

## Features

- 4 parallel dimensions (Prime, Inverted, Void, Nexus)
- Fracture events that swap chunks between dimensions
- Dimensional anchors to protect critical areas
- Phase shifts for inter-dimensional travel
- Fracture sickness from rapid dimension changes
- Stability energy system powering all tech
- Resource transmutation across dimensions
- 10 creatures adapted to dimensional instability

## Quick Start

```bash
cargo build
cargo run
```

Built on the [Lattice](https://github.com/sudokatie/lattice) survival engine.
